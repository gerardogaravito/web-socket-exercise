# Web Socket Exercise ^.^

website url: https://web-socket-exercise.vercel.app/

### Run the project

- Clone the project
  `git clone [url]`

- Install dependencies
  `npm install`

- Run develop enviroment
  `npm run dev`

- Build production project
  `npm run build`

- Run production project
  `npm run start`

### Tools
